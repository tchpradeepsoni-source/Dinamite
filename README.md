# 🚀 D66 Language (Version 1)

A simple programming language that runs directly in the browser.

👉 Built as a learning project to understand how programming languages work.

---

## 🌟 Features

- Variables
- Maths operations (+, -, *, /)
- Print output
- If / Else logic
- Functions
- Loops (repeat, while)
- Error handling
- Runs in browser (no install needed)
